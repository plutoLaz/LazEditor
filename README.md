# LazEditor

Eine Editor Komponente für Lazarus.

Es sollen folgende Formatierungen unterstützt werden:
- Fett
- kursiv
- Unterstrichen
- Durchgestrichen
- Vordergrund Farbe
- Hintergrund Farbe
- Schrift Größen
- Schrift Name
- Zeilen Ausrichtungen(Links, Zentriert, Rechts, Block)
- Paragraph
- Grafiken(Auch im Fließtext)
- Tabellen
